## Personal website

Built with [Eleventy](https://11ty.dev).

## Write

```bash
npm run new
```

## Run locally

```bash
npm i -g vercel
pnpm i
pnpm start
```

## Build locally

```bash
npm i -g turbo
turbo build
```

## Deploy

The `main` branch is auto deployed by Vercel:

https://vercel.com/mehul-kar/mehulkar
