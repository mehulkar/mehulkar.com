# mehulkar.com

My personal website. Built with [Eleventy](https://11ty.dev).

## Write

```bash
pnpm new
```

## Run locally

```bash
pnpm i
pnpm add -g vercel
vc dev
```

## Build locally

```bash
npm i -g turbo
turbo build
```

## Deploy

The `main` branch is auto deployed by Vercel:

https://vercel.com/mehul-kar/mehulkar
